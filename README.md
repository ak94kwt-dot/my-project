# دورة n8n من الصفر — تأسيس كامل حتى بناء AI Agent 🚀

أهلاً! هذي دورة تعليمية كاملة بالعربي تأخذك من الصفر في n8n حتى تبني ايجنت ذكاء اصطناعي متكامل بنفسك.

## شنو يميز هالدورة؟

كل درس مرتبط بـ **workflow حقيقي مبني داخل حسابك في n8n** — مو بس قراءة، تفتح الـ workflow وتجربه وتعدل عليه بيدك.

حسابك: [almearaj1ai.app.n8n.cloud](https://almearaj1ai.app.n8n.cloud)

## مسار التعلم المقترح

1. **اليوم الأول**: اقرأ درس 1 و 2، وافتح workflow "درس 2" ونفّذه أكثر من مرة وغيّر القيم.
2. **اليوم الثاني**: درس 3 و 4 و 5 — افتح workflow "درس 5" وشوف كيف الشرط IF يوجه البيانات.
3. **اليوم الثالث**: درس 6 و 7 — أضف مفتاح OpenAI API وجرب أول ايجنت لك.
4. **اليوم الرابع**: درس 8 — كلّم الايجنت المتكامل واسأله عن الطقس، وبعدين عدّل عليه وأضف أداة جديدة.
5. **اليوم الخامس**: درس 9 — انشر أول workflow وتعلم تعالج الأخطاء.

> ⚠️ **مهم لدروس 7 و 8**: الايجنت يحتاج مفتاح OpenAI API. طريقة إضافته مشروحة في [درس 7](docs/07-ai-agent.md#إضافة-مفتاح-api).
