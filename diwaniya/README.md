# ناقشنا — حزمة المنتج

## شنو هذا؟
منتج خليجي فيه طبقتان:
- **المستشار الشخصي** (الرئيسية): أسئلة يومية خفيفة تبني ملفاً خاصاً غير قابل للنشر، يتعلّم منه الذكاء الاصطناعي أنماط شخصيتك ويعطيك نصائح.
- **الفريج الحيّ** (أداة): مشهد SVG فيه ٢٤ شخصية تتناقش بوجوه متغيّرة وعدوى مزاجية، وتتوقّع ردة فعل المجتمع على محتواك قبل النشر (مع تحليل: منو أثّر على منو ومنو غيّر رأيه).

## بنية المشروع

| الملف | شنو يسوي |
|------|----------|
| `engine.js` | **المحرّك القواعدي** — ٢٤ شخصية + رادار ٦ أبعاد + منطق الحكم (fallback). |
| `index.html` | الصفحة الكاملة (تُعرض على `/` لايف). |
| `app.js` | يبني الموقع ديناميكياً: المستشار + لوحة النتائج + التحليل. |
| `questions.js` · `profile.js` · `advisor.css` | المستشار الشخصي: بنك الأسئلة + الملف المحلي (localStorage) + الأنماط. |
| `freej.js` · `freej.css` | الفريج الحيّ: المشهد + النقاش + العدوى المزاجية + التحليل. |
| `api/generate.js` | سيرفرليس: توقّع ردة الفعل عبر Claude (مع fallback للمحاكي). |
| `api/profile-insights.js` | سيرفرليس: استخلاص أنماط الشخصية من الملف الخاص. |
| `api/debate-narrate.js` | سيرفرليس: سرد تحليل النقاش بأسلوب أحلى (اختياري). |
| `docs/engine-spec-v4.6.md` | الوثيقة الكاملة — القواعد الذهبية، قاموس اللهجات، حلقة التعلّم. |
| `docs/NOTES.md` · `docs/NOTES-2.md` | مرجع التطوير الجاري والباكلوگ. |
| `docs/partner-deck.pptx` | عرض الشريك المؤسس. |

## التشغيل محلياً

```bash
cd diwaniya
python3 -m http.server 8000   # افتح http://localhost:8000/
```

اختبار المحرّك بالـ Terminal:
```bash
node -e "const {generate}=require('./engine.js'); console.log(JSON.stringify(generate('شريت سيارة جديدة وعرضتها قدام الناس'),null,2));"
```

## الذكاء الحقيقي (Claude)
الواجهة تنادي `/api/*` أولاً، وإذا فشل أي شي ترجع للمحاكي القواعدي تلقائياً — التجربة ما تنكسر.
لتفعيل Claude: عيّن `ANTHROPIC_API_KEY` ببيئة Vercel (لا يُخزّن بالكود أبداً) ثم Redeploy.

## للفريق التقني
- البرومبتات بـ `api/*.js`، والمعايرة (عتبات الخطر، التصعيد) موسومة بـ`engine.js`.
- حدود المحتوى الحسّاس: «نصوّر النار، ما نشعلها» — نتوقّع ونحذّر من الإساءة، ما ننتجها.
- خصوصية الملف الشخصي: محلي بالمتصفّح، والإجابات الحسّاسة (دين/سياسة/صحة عميقة) لا تُرسل للسيرفر.
