## الحالة الحالية
- المشروع مدموج على `main`، منشور لايف على Vercel (Git Integration، أي push على main ينشر تلقائياً).
- الرابط: `https://my-project-git-main-ak-s-pr.vercel.app`
- جسر Claude الحقيقي جاهز بالكامل: `diwaniya/api/generate.js` (سيرفرليس، Anthropic Messages API، نموذج claude-sonnet-4-6، Prompt Caching، fallback تلقائي لـ`engine.js` لو فشل).
- **الفريج الحيّ — M1 منجَز ومنشور:** `freej.js` + `freej.css` — خريطة SVG كاملة، 24 شخصية متحركة، كوريوغرافيا (تجمّع → إضاءة مبانٍ → فقاعات كلام متتابعة → HUD حكم بلون الفيردكت). صار **أداة** تُفتح من زر «أدوات».
- **المستشار الشخصي — A1 منجَز:** الصفحة الرئيسية الجديدة. أقسام على اليمين + «سؤال اليوم» خفيف يبني **ملفاً خاصاً غير قابل للنشر** (localStorage) → استخلاص أنماط الشخصية + نصائح (عبر `/api/profile-insights` مع fallback محلي). الملفات: `questions.js` (بنك أسئلة بعيد عن السياسة/الدين/الصحة العميقة) · `profile.js` (التخزين + محرك السؤال + الاستخلاص) · `api/profile-insights.js` (سيرفرليس Claude) · `advisor.css`.

## معلّق على المالك (لا تنفّذ بدون طلبه)
1. **🔑 "المفتاح"** — ANTHROPIC_API_KEY:
   1) console.anthropic.com → Settings → API Keys → Create Key
   2) Settings → Billing → شحن $5–10
   3) Vercel → my-project → Settings → Environment Variables → `ANTHROPIC_API_KEY` = المفتاح → Production → Save
   4) Vercel → Deployments → آخر نشر → ··· → Redeploy
   > بدونه: الموقع يعمل لكن بردود المحاكي القواعدي (مكرّرة).
2. **تعطيل Vercel Deployment Protection** (Settings → Deployment Protection → Disabled → Save) — بدونه الموقع يطلب تسجيل دخول لأي زائر عام.
