# الدرس 4: المشغلات Triggers

الـ Trigger هو أول نود في أي workflow — يحدد **متى** يشتغل. بدون trigger ما في workflow.

## أنواع الـ Triggers الأساسية

### 1. Manual Trigger — للتجربة والتعلم
يشتغل لما تضغط زر **Execute workflow**. استخدمناه في دروس 2 و 5.
- ممتاز وقت البناء والتجربة
- ما يصلح للأتمتة الحقيقية (محد بيضغط الزر كل يوم!)

### 2. Schedule Trigger — للمهام الدورية
يشتغل تلقائياً بجدول تحدده:
- كل X دقيقة / ساعة / يوم / أسبوع / شهر
- أو بصيغة **Cron** للجداول المعقدة (مثلاً: "كل أحد وثلاثاء الساعة 9:30")

أمثلة استخدام: تقرير يومي، نسخ احتياطي أسبوعي، فحص أسعار كل ساعة.

> ⚠️ Schedule Trigger ما يشتغل إلا إذا كان الـ workflow **منشور (Active)** — شوف درس 9.

### 3. Webhook — للأحداث الخارجية
يعطيك **رابط URL خاص**، وأي برنامج خارجي يرسل طلب لهذا الرابط → الـ workflow يشتغل فوراً ومعه البيانات المرسلة.

أمثلة: موقعك يرسل بيانات فورم تسجيل، تطبيق دفع يبلغك بعملية جديدة، GitHub يبلغك بـ commit جديد.

البيانات توصل عادة داخل حقل `body`:
```
{{ $json.body.email }}
```

### 4. Chat Trigger — للمحادثات والايجنت 🤖
يفتح **واجهة محادثة** جاهزة من n8n. كل رسالة يرسلها المستخدم تشغل الـ workflow، ونص الرسالة يوصل في:
```
{{ $json.chatInput }}
```
هذا اللي نستخدمه مع AI Agent في دروس 7 و 8.

### 5. App Triggers — مشغلات التطبيقات
نودات جاهزة تراقب خدمات معينة: إيميل جديد في Gmail، رسالة في تيليجرام، صف جديد في Google Sheets... كلها تحتاج ربط حساب (درس 6).

## أي trigger أختار؟

| الحالة | الـ Trigger |
|--------|------------|
| أجرب وأتعلم | Manual |
| مهمة دورية بوقت محدد | Schedule |
| برنامج خارجي يرسل لي بيانات | Webhook |
| محادثة / ايجنت AI | Chat Trigger |
| أراقب خدمة معينة (إيميل، تيليجرام...) | App Trigger الخاص بها |

## معلومة مهمة: ممكن أكثر من trigger بنفس الـ workflow

تقدر تحط مثلاً Webhook و Schedule Trigger في نفس الـ workflow — كل واحد يشغّل مساره، وكل تنفيذ مستقل تماماً عن الثاني.

## تمرين الدرس

1. افتح workflow "درس 5" واستبدل فكرياً (بدون تعديل): لو نبي تقرير الطقس يجي **كل يوم الساعة 7 الصبح** — أي trigger نستخدم وشنو إعداداته؟
2. افتح workflow "درس 7" واضغط على نود "محادثة - Chat Trigger" وشوف إعداداته: شنو عنوان المحادثة؟ وشنو الرسالة الافتتاحية؟
