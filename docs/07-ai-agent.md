# الدرس 7: الذكاء الاصطناعي — AI Agent 🤖

> **الـ Workflow الجاهز**: [درس 7 - أول AI Agent](https://almearaj1ai.app.n8n.cloud/workflow/FWRcecsnoctQMuYy)

وصلنا لقلب الدورة! في هذا الدرس تفهم كيف تبني ايجنت ذكاء اصطناعي يحاورك ويستخدم أدوات.

## شنو الفرق بين Chatbot عادي و Agent؟

- **Chatbot عادي**: ياخذ سؤالك → يرسله للموديل → يرجع الجواب. بس.
- **Agent (ايجنت)**: ياخذ سؤالك → **يفكر** → **يقرر** إذا يحتاج يستخدم أداة (آلة حاسبة، بحث، API...) → يستخدمها → يفكر بالنتيجة → يجاوبك. ممكن يستخدم عدة أدوات ورا بعض لحل مشكلة وحدة.

## تشريح الايجنت في n8n

افتح الـ workflow وشوف — الايجنت يتكون من نود رئيسي و"نودات فرعية" (sub-nodes) متعلقة فيه من تحت:

```
[Chat Trigger] ──────→ [AI Agent]
                        │ │ │
              ┌─────────┘ │ └──────────┐
        [OpenAI Model] [Memory]  [Calculator]
            العقل        الذاكرة      أداة
```

| القطعة | النود | وظيفتها |
|--------|------|---------|
| واجهة المحادثة | Chat Trigger | يستقبل رسائل المستخدم ويرجع الردود |
| المنسق | AI Agent | يدير كل شي: يستقبل، يفكر، يستخدم الأدوات، يرد |
| العقل | OpenAI Chat Model | الموديل اللغوي اللي يفكر ويكتب (يحتاج مفتاح API) |
| الذاكرة | Simple Memory | يحفظ آخر 10 رسائل عشان الايجنت يتذكر سياق المحادثة |
| الأداة | Calculator | الايجنت يستدعيها لما يحتاج يحسب |

## الـ System Message — شخصية الايجنت

اضغط مرتين على نود "الايجنت - AI Agent" وشوف خانة **System Message**:

> "أنت مساعد ذكي ودود تتكلم بالعربي. عندك أداة اسمها Calculator - استخدمها لأي عملية حسابية..."

هذي **التعليمات الدائمة** للايجنت — تحدد شخصيته ولغته وقواعده وأدواته. أهم قواعد كتابتها:
1. حدد **الهدف**: شنو وظيفة الايجنت؟
2. اذكر **الأدوات بأسمائها** ومتى يستخدم كل وحدة
3. حدد **أسلوب الرد**: مختصر؟ رسمي؟ بالعربي؟

## إضافة مفتاح API

قبل ما تجرب، الايجنت يحتاج عقل! خطوات إضافة مفتاح OpenAI:

1. افتح الـ workflow → اضغط مرتين على نود **"العقل - OpenAI Model"**
2. في خانة **Credential to connect with** اضغط واختر **Create new credential**
3. الصق مفتاحك (من [platform.openai.com/api-keys](https://platform.openai.com/api-keys)) في خانة **API Key**
4. **Save** ✅

> 💡 إذا تفضل موديل غير OpenAI (مثل Claude من Anthropic أو Gemini من Google): احذف نود OpenAI Model، أضف نود الموديل البديل (مثل Anthropic Chat Model)، واسحب خط من مدخل **Chat Model*** تحت نود الـ Agent إليه. كل الباقي يبقى نفسه.

## جرب الايجنت!

1. اضغط زر **Open chat** (تحت في الـ canvas)
2. اكتب: **"كم ناتج 1234 ضرب 5678؟"**
3. لاحظ في الـ canvas: نود Calculator اشتغل! الايجنت قرر بنفسه إنه يحتاج الحاسبة
4. اكتب: **"واضرب الناتج في 2"** — بيتذكر الناتج السابق لأن عنده **ذاكرة**
5. اسأله سؤال عام: "شنو عاصمة اليابان؟" — ما بيستخدم الحاسبة لأنه ما يحتاجها

## ليش الذاكرة مهمة؟

بدون نود Memory، كل رسالة تكون **محادثة جديدة** — الايجنت ما يدري شنو قلت له قبل ثانية! الـ Simple Memory يحفظ آخر X رسائل (عندنا 10) ويعطيها للموديل مع كل سؤال جديد.

## تمارين الدرس

1. غيّر الـ System Message وخل الايجنت يرد دايماً بأسلوب فكاهي — جرب الفرق.
2. غيّر **Context Window Length** في نود الذاكرة من 10 إلى 2، وسولف معاه شوي — متى بدأ "ينسى"؟
3. أضف أداة ثانية: اضغط + على مدخل **Tool** تحت الـ Agent وأضف **Think Tool** — أداة تخلي الايجنت يفكر بصوت عالي قبل الرد.
