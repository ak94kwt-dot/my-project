# الدرس 3: البيانات والتعبيرات Expressions

هذا أهم درس نظري بالدورة — إذا فهمته، انفتح لك 80% من n8n.

## كيف تنتقل البيانات؟ (Items و JSON)

البيانات في n8n تمشي بين النودات على شكل **items**، وكل item هو **JSON**:

```json
{
  "studentName": "أحمد",
  "lessonNumber": 2,
  "isActive": true,
  "scores": [90, 85, 77],
  "address": {
    "city": "الكويت",
    "area": "السالمية"
  }
}
```

- النص بين `" "` → **string**
- الأرقام بدون أقواس → **number**
- `true` / `false` → **boolean**
- `[ ]` → **array** (قائمة)
- `{ }` → **object** (كائن متداخل)

## التعبيرات Expressions — قوة n8n الحقيقية

التعبير هو كود صغير بين قوسين مزدوجين `{{ }}` يقرأ بيانات من نودات سابقة **وقت التنفيذ**.

### `$json` — بيانات النود السابق مباشرة

```
{{ $json.studentName }}          ← يقرأ حقل studentName
{{ $json.address.city }}         ← يقرأ حقل متداخل (city داخل address)
{{ $json.scores[0] }}            ← أول عنصر بالقائمة (الترقيم يبدأ من 0)
```

في workflow درس 5 استخدمنا بالضبط هالشي لاستخراج درجة الحرارة من رد الـ API:
```
{{ $json.current_weather.temperature }}
```

### `$('اسم النود')` — بيانات أي نود سابق بالاسم

أحياناً تحتاج بيانات من نود مو اللي قبلك مباشرة:
```
{{ $('إنشاء بيانات - Set').item.json.studentName }}
```

### متغيرات جاهزة مفيدة

| التعبير | معناه |
|---------|-------|
| `{{ $now }}` | الوقت والتاريخ الحين |
| `{{ $now.toFormat('yyyy-MM-dd') }}` | التاريخ بصيغة معينة |
| `{{ $today }}` | بداية اليوم |
| `{{ $workflow.name }}` | اسم الـ workflow |
| `{{ $execution.id }}` | رقم التنفيذ الحالي |

### دمج النص مع التعبيرات

تقدر تخلط نص ثابت مع تعبيرات:
```
أهلاً {{ $json.studentName }}! درجة الحرارة اليوم {{ $json.temperature }}
```

وتقدر تكتب شروط داخل التعبير (ternary):
```
{{ $json.temperature > 35 ? "حار" : "لطيف" }}
```

## جرب بنفسك في n8n

1. افتح [workflow درس 2](https://almearaj1ai.app.n8n.cloud/workflow/nrBMQMSXw5DHXKO2)
2. اضغط مرتين على نود "تكوين رسالة - Expressions"
3. اضغط على حقل `welcomeMessage` — لاحظ القيمة فيها `{{ $json.studentName }}`
4. لاحظ تحت الحقل في **نتيجة مباشرة (Result)** تتحدث وأنت تكتب — هذي أفضل طريقة تجرب التعبيرات

## قاعدة ذهبية

> أي شي تبيه **ديناميكي** (يتغير حسب البيانات) لازم يكون داخل `{{ }}`.
> أي شي **ثابت** اكتبه عادي بدون أقواس.
