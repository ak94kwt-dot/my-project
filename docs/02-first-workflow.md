# الدرس 2: أول Workflow لك 🎉

> **الـ Workflow الجاهز**: [درس 2 - أول Workflow](https://almearaj1ai.app.n8n.cloud/workflow/nrBMQMSXw5DHXKO2)

في هذا الدرس بتنفذ أول workflow وبتفهم كيف البيانات تتحرك بين النودات.

## مكونات الـ Workflow

افتح الـ workflow وبتلقى 3 نودات:

```
[ابدأ هنا - Manual Trigger] ──→ [إنشاء بيانات - Set] ──→ [تكوين رسالة - Expressions]
```

| النود | النوع | وظيفته |
|------|------|--------|
| ابدأ هنا | Manual Trigger | نقطة البداية — يشتغل لما تضغط الزر |
| إنشاء بيانات | Edit Fields (Set) | ينشئ 3 حقول: اسم الطالب، اسم الدورة، رقم الدرس |
| تكوين رسالة | Edit Fields (Set) | يركّب رسالة ترحيب من الحقول السابقة |

## نفّذها بنفسك — خطوة بخطوة

1. افتح الـ workflow من الرابط فوق
2. اضغط زر **Execute workflow** (تحت في النص)
3. لاحظ النودات صارت خضراء وحدة ورا الثانية ✅
4. **اضغط على نود "إنشاء بيانات - Set"** — بتلقى لوحة فيها:
   - **INPUT** (يسار): شنو دخل للنود
   - **OUTPUT** (يمين): شنو طلع منه
5. شوف الـ OUTPUT — بتلقى:
   ```json
   {
     "studentName": "أحمد",
     "courseName": "دورة n8n من الصفر",
     "lessonNumber": 2
   }
   ```
6. الحين اضغط على نود "تكوين رسالة" وشوف كيف نفس البيانات دخلت له، وطلعت ومعها حقل جديد `welcomeMessage`

## نود Set — أهم نود للمبتدئ

نود **Edit Fields (Set)** يخليك تضيف أو تعدل حقول البيانات. كل حقل له:
- **Name**: اسم الحقل (مثل `studentName`)
- **Type**: نوعه (string نص / number رقم / boolean صح أو خطأ...)
- **Value**: قيمته — ممكن قيمة ثابتة أو **تعبير** يقرأ من بيانات سابقة (درس 3)

## تمارين الدرس (مهمة!)

1. **غيّر الاسم**: اضغط مرتين على نود "إنشاء بيانات"، غيّر قيمة `studentName` لاسمك أنت، ونفّذ من جديد. شوف الرسالة تغيرت؟
2. **أضف حقل جديد**: في نفس النود اضغط **Add Field** وأضف حقل `city` نوعه string وقيمته مدينتك. نفّذ وشوفه في الـ OUTPUT.
3. **تحدي**: خل الرسالة في نود "تكوين رسالة" تذكر المدينة بعد. تلميح: استخدم `{{ $json.city }}` (بتفهم معناها بالدرس الجاي).

- ✅ كيف تقرأ INPUT و OUTPUT لأي نود
- ✅ نود Set لإنشاء وتعديل الحقول
