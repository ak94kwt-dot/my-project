# الدرس 8: مشروع عملي — ايجنت متكامل بأدوات حقيقية 🚀

> **الـ Workflow الجاهز**: [درس 8 - ايجنت متكامل](https://almearaj1ai.app.n8n.cloud/workflow/VO19bMDUfnCdo1iU)

هذا المشروع الختامي: ايجنت طقس ذكي عنده **3 أدوات** ويقرر بنفسه متى يستخدم كل وحدة.

## شنو يسوي هالايجنت؟

اسأله "شلون الجو في الكويت؟" وهو:
1. **يفكر** (Think Tool): المستخدم يبي طقس الكويت، إحداثياتها 29.37, 47.97
2. **يستدعي API حقيقي** (Weather Tool): يرسل طلب لـ Open-Meteo بالإحداثيات
3. **يفسر** النتيجة ويرد عليك بالعربي مع نصيحة

## البنية

```
[Chat Trigger] ──────→ [AI Agent]
                       │ │ │ │ │
         ┌─────────────┘ │ │ │ └────────────────┐
   [OpenAI Model]  [Memory] │ [Calculator]  [Think Tool]
                            │
                     [Weather Tool]
                  (HTTP Request Tool)
```

## النجمة هنا: HTTP Request Tool مع fromAI ⭐

افتح نود **"أداة الطقس - Weather Tool"** وشوف الـ Query Parameters:

- `latitude` → قيمتها `$fromAI('latitude', '...')`
- `longitude` → قيمتها `$fromAI('longitude', '...')`

شنو معنى `fromAI`؟ معناها: **"يا ايجنت، أنت حدد هالقيمة بنفسك"**.

لما تسأل عن دبي، الايجنت نفسه يطلع إحداثيات دبي ويمررها للأداة. ما كتبنا ولا سطر كود لتحويل "دبي" لإحداثيات — الموديل يعرفها!

هذا المفهوم هو **أهم فكرة في بناء أدوات الايجنت**: أنت تعرّف الأداة وحدودها، والايجنت يعبي الفراغات حسب سياق المحادثة.

## الـ System Message هنا أطول وأدق

اضغط على نود الـ Agent وشوف — التعليمات تشرح **كل أداة بالاسم ومتى تستخدم**:

> - Weather Tool: تجيب حالة الطقس... مرر latitude و longitude
> - Calculator: لأي عملية حسابية
> - Think: للتفكير خطوة بخطوة قبل الأسئلة المعقدة

كل ما كانت تعليمات الأدوات أوضح، كان قرار الايجنت أدق.

## جربه! (يحتاج مفتاح OpenAI من درس 7)

أسئلة تدريجية بالصعوبة:

1. **"شلون الجو في الكويت؟"** → أداة وحدة (Weather)
2. **"قارن الجو بين الكويت ودبي"** → أداتين (Weather مرتين!)
3. **"كم الفرق بالحرارة بين الرياض والقاهرة، وكم يساوي الفرق بالفهرنهايت؟"** → ثلاث أدوات (Weather ×2 + Calculator)

بعد كل سؤال، ارجع للـ canvas وشوف أي نودات اشتغلت (الأرقام الخضراء فوق الأدوات تبين عدد مرات الاستدعاء).

## أفكار لتطوير المشروع (هذا هو التعلم الحقيقي!)

1. **أداة عملات**: أضف HTTP Request Tool ثاني على `https://api.frankfurter.app/latest?from=USD&to=KWD` وخل `from` و `to` بـ fromAI — صار عندك ايجنت يحول العملات!
2. **ذاكرة دائمة**: استبدل Simple Memory بقاعدة بيانات (مثل Postgres Memory) عشان الايجنت يتذكر حتى بعد إعادة التشغيل.
3. **اربطه بتيليجرام**: استبدل Chat Trigger بـ Telegram Trigger وأضف نود Telegram للرد — ايجنتك صار بوت تيليجرام حقيقي.
4. **ايجنت متخصص لشغلك**: غيّر الأدوات والـ System Message حسب احتياجك — خدمة عملاء، ملخص أخبار، مساعد دراسة...
